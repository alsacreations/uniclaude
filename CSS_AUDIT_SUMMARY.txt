🔴 PROBLÈMES CRITIQUES (À corriger en priorité)

1. ❌ Architecture monolithique (484 lignes dans 1 fichier)
   → Découper en modules (config/ theme/ components/ utilities/)

2. ❌ Pas de Stylelint
   → Installer stylelint + stylelint-order + prettier

3. ❌ Système de design non conforme
   → Architecture 3 niveaux manquante :
      • Primitives (--color-*, --spacing-*, --text-*)
      • Tokens (--surface, --on-surface, --primary)
      • Composants (utilisation des tokens)

4. ❌ Pas de Bretzel Layouts
   → Remplacer Grid/Flexbox manuels par data-layout="..."

═══════════════════════════════════════════════════════════════

🟡 PROBLÈMES IMPORTANTS (À améliorer)

5. ⚠️ Nommage BEM incohérent
   Exemples : .subtitle, .stats, .filters
   → Devrait être : .header-subtitle, .search-stats, .search-filters

6. ⚠️ Ordre SMACSS non respecté
   Propriétés CSS dans le désordre
   → Installer stylelint-order pour tri automatique

7. ⚠️ Pas de nesting CSS natif
   État actuel : Sélecteurs séparés (.card:hover { })
   → Utiliser : .card { &:hover { } }

8. ⚠️ Breakpoints non standardisés
   État actuel : @media (max-width: 768px) { }
   → Guidelines : @media (width >= 48rem) { } (mobile-first)

9. ⚠️ Pas de dark/light mode
   → Implémenter light-dark() et color-scheme

🎯 PLAN D'ACTION RECOMMANDÉ

Phase 1 : OUTILLAGE (1-2h)
  □ Installer Stylelint + Prettier
  □ Configurer stylelint.config.js
  □ Créer scripts npm (lint, format)

Phase 2 : ARCHITECTURE (4-6h)
  □ Découper styles.css en modules
  □ Créer theme.css (primitives)
  □ Créer theme-tokens.css (tokens)
  □ Mettre en place @layer

Phase 3 : LAYOUTS BRETZEL (2-3h)
  □ Installer Bretzel Layouts
  □ Remplacer .characters-grid par data-layout="autogrid"
  □ Remplacer .filters par data-layout="cluster"
  □ Utiliser data-layout="stack" où pertinent

Phase 4 : AMÉLIORATIONS (2-3h)
  □ Implémenter light-dark() pour dark mode
  □ Standardiser nommage BEM
  □ Passer en mobile-first
  □ Utiliser nesting CSS natif

TEMPS TOTAL ESTIMÉ : 10-15 heures
